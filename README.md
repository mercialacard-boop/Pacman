# PAC-MAN - Schulversion

Classic Pac-Man game. Built with vanilla HTML5 Canvas. Built with vanilla HTML5 Canvas.

## Play

Open `index.html` in any browser. No build step, no dependencies.

## Controls

| Device | Control |
|--------|---------|
| Desktop | WASD or Arrow Keys, Enter to start |
| Mobile | D-Pad buttons, swipe gestures, or tap Start |
| Mobile (portrait) | Rotate device to landscape |

## Deploy on Render

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) and click **New > Static Site**
3. Connect your GitHub repo
4. Settings:
   - **Build Command:** *(leave empty)*
   - **Publish Directory:** `.`
5. Click **Deploy** — done, live URL in ~30 seconds

## Deploy on GitHub Pages

1. Push to GitHub
2. Go to repo **Settings > Pages**
3. Source: **Deploy from a branch**
4. Branch: `main`, folder: `/ (root)`
5. Save — live at `https://yourusername.github.io/pacman-schulversion`
